# 카카오페이 사전 과제 주택금융API개발

### 0. 개발 환경 및 사용 라이브러리
- Java 8
- SpringBoot 2.1
- SpringSecurity 5.1
- JPA
- H2
- jjwt
- swagger2
- opencsv
- commons-math3
- lombok
- SpringToolSuite4

### 1. 빌드 및 실행

```
$ git clone https://github.com/bbjjy/kakaopay-bjy.git
$ cd kakaopay-bjy
$ ./gradlew clean build
$ java -jar build/libs/kakaopay-bjy-0.0.1-SNAPSHOT.jar
```
